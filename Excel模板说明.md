# 五险一金计算器 - Excel模板说明

## 文件说明

### 城市标准文件
- **cities-template.xlsx** (英文列名)
- **城市标准模板.xlsx** (中文列名)

必需字段：
- city_name / 城市名: 城市名称
- year / 年份: 年份 (格式: 2024)
- base_min / 基数下限: 社保缴费基数下限
- base_max / 基数上限: 社保缴费基数上限
- rate / 费率: 公司承担的综合费率

### 员工工资文件
- **salaries-template.xlsx** (英文列名)
- **员工工资模板.xlsx** (中文列名)

必需字段：
- employee_id / 员工ID: 员工唯一标识
- employee_name / 员工姓名: 员工姓名
- month / 月份: 年份月份 (格式: YYYYMM, 如 202401)
- salary_amount / 工资金额: 当月税前工资金额

## 注意事项

1. **不要包含id列** - 数据库会自动生成
2. 月份必须是6位数字格式 (YYYYMM)
3. 所有金额字段使用数字，不要包含货币符号
4. 费率使用小数形式 (如 0.17 表示 17%)
